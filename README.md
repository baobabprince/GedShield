# GedShield 🛡️ — Privacy Shield & Topology Analyzer for GEDCOM Family Trees

**GedShield** is a web-based, privacy-first management and analysis platform for GEDCOM (`.ged`) family trees. It parses, anonymizes, edits, and visualizes family files completely client-side.

[עברית בהמשך]

---

### Features | תכונות עיקריות
- **Privacy Engine**: Anonymize or delete living members, remove disconnected branches, and clean up identifiers to be GDPR-compliant.
- **Topological Analysis**: Computes graph metrics, detects pedigree collapse (אובדן אילן יוחסין), and analyzes generational stratification asynchronously via Web Workers.
- **Interactive Visualizer**: Drag-and-drop, zoom, pan, and search family nodes on a force-directed graph (supporting Light and Dark modes).
- **GEDCOM Compliant**: Export clean physical files adhering to standard GEDCOM formats.

### עברית
**GedShield** הוא כלי מבוסס דפדפן המותאם לשמירה על פרטיות אילנות יוחסין (קובצי GEDCOM). הכלי מאפשר לטעון, לערוך, לבצע אנונימיזציה, לנתח ולהציג ויזואלית את עץ המשפחה באופן מקומי ומאובטח לחלוטין.

- **מגן פרטיות**: מחיקה או טשטוש נתונים של אנשים חיים, ניקוי ענפים מנותקים והתאמה לתקני פרטיות.
- **ניתוח טופולוגי**: זיהוי חזרתיות גנטית/אובדן אילן יוחסין (Pedigree Collapse), חישוב מרחקים דוריים ומדדים שונים ביעילות (ע"י Web Worker).
- **תצוגה אינטראקטיבית**: גרף כוחות דו-ממדי דינמי עם תמיכה במצב יום ולילה, חיפוש מהיר וסינון קל.
